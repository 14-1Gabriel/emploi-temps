# 📅 Emploi du temps - Gestionnaire de planning

Application web de gestion d'emploi du temps avec rappels automatiques.

## 🚀 Fonctionnalités

- ✅ Inscription et connexion sécurisées
- ✅ Création, modification et suppression d'activités
- ✅ Vue hebdomadaire des cours
- ✅ Rappels 15 minutes avant chaque activité
- ✅ Interface responsive

## 🛠️ Technologies

- PHP 7.4+
- MySQL / MariaDB
- HTML5/CSS3
- JavaScript
- Vercel (hébergement)
